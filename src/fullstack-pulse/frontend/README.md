## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Environment Variables

Create `.env` file:

```

VITE_CONTRACT_ADDRESS=0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb
```
